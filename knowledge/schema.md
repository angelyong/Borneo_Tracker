# Static Knowledge Record Schema

Phase 1 static knowledge files contain arrays of records with this shape:

```json
{
  "id": "forest-cover",
  "title": "Forest Cover",
  "category": "environmental",
  "content": "",
  "regions": [],
  "relatedSdgs": [],
  "unit": null,
  "sourceName": "",
  "sourceUrl": "",
  "pageUrl": "/esg",
  "updatedAt": null,
  "status": "placeholder"
}
```

Records marked `placeholder` are safe implementation scaffolding, not authoritative facts or statistics.
